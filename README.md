# 📱 Course Companion App

## About

This is a mobile app built as part of a Mobile Programming course project. The idea is to help students keep track of their courses, assignments, and deadlines in one place.

## Features

* add and remove courses
* track assignments and deadlines
* mark tasks as completed
* notes for each course
* simple search

## Tech

* Kotlin / Java / Flutter / React Native
* SQLite / Room

## Run

1. clone the repo
2. open in Android Studio / VS Code
3. run on emulator or device

## Note

This project was made for learning purposes, focusing on basic mobile app functionality.
